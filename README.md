<h3>📐 A powerful console tool for solving analytical geometry problems</h3>

<p>
Interactively work with points, lines, and geometric objects —  
compute intersections, construct lines, analyze triangles, and more.
</p>

</div>

---

## 🚀 Features

✨ Check if a point lies on a line  
✨ Construct parallel and perpendicular lines  
✨ Compute intersections of lines  
✨ Work with triangles (medians, altitudes, bisectors)  
✨ Tangents to parabolas  
✨ Line–parabola intersections  
✨ Classify quadrilaterals formed by lines  

---

## 🧠 Capabilities Overview

### 📍 Points & Lines
- Define custom points and lines  
- Validate input with robust parsing  
- Generate clean equation output  

### 📏 Geometry Operations
- Parallel & perpendicular constructions  
- Intersection detection  
- Collinearity checks  

### 🔺 Triangle Geometry
- Medians  
- Altitudes  
- Perpendicular bisectors  

### 📈 Advanced
- Tangents to parabolas  
- Intersection with quadratic functions  
- Quadrilateral classification:
  - Parallelogram  
  - Rectangle  
  - Trapezoid
